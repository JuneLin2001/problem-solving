## 解題思路

Discussion 有個人貼了所有的 ["beautiful" numbers 的清單](https://leetcode.com/problems/next-greater-numerically-balanced-number/description/comments/3211016/)  
所以就用 for 迴圈找出最小且符合條件的數  
對

```js
for (let i of beautifulNumbers) {
  if (i > n) {
    return i;
  }
}
```
