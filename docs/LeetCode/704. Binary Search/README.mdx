## 解題思路

來練習一下經典題，試了兩種解法分別別放在 js 和 ts 。  
js 是從頭開始暴力遍歷陣列的 Linear Search，ts 則是效能較好的 Binary Search。  
而題目描述有提到 `You must write an algorithm with O(log n) runtime complexity.`  
所以真正的正解會是 Binary Search 的作法。
