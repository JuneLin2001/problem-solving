## 解題思路

想法是使用 counter 來記錄數字出現的數量

```js
for (let i = 0; i < nums.length; i++) {
  if (counter[nums[i]]) {
    counter[nums[i]]++;
  } else {
    counter[nums[i]] = 1;
  }
}
```

遍歷一遍陣列後再次遍歷 `counter`，若出現兩次則記錄起來，最後回傳答案。

```js
for (let key in counter) {
  if (counter[key] === 2) {
    result.push(parseInt(key));
  }
}
```
