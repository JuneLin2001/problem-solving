### 解題思路

把題目描述轉成 JS 程式碼，先是遍歷迴圈，當遇到該格是 `1` 的時候就檢查往前 k 格之內有沒有其他的 `1`，若有則 `false`，若遍歷完一遍後皆沒有則 `true`。

### 心得

我的解法用了兩個 for 迴圈，時間複雜度變成 O($n^2$) 不太好，應該可以再簡化
