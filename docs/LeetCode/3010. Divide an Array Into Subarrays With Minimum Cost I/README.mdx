## 解題思路

因為 nums 的第一個項目一定會包含在最終答案之中，所以第一步就是先加上 `nums[0]`  
接著移除 `nums[0]` 後，再對剩下來的進行排序  
排序後把最小和第二小再加進 `result` 就是答案了！
